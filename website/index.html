Github Notifier

Invite GitHub

A Discord bot emulating GitHub webhooks

But why?

Let's say you like someone else's project, and it's hosted
on GitHub. You also use Discord, and would like to get
notified on new releases. Then a webhook would be the
ideal solution, except you're not the owner of that
repository, so you use this bot!

How it works?

Simple setup

Just invite the bot and use git -sc
to set the default channel for notifications

But how do I add repositories?

With git -a https://github.com/username/repository,
the bot automatically checks if the url is a valid GitHub
url, and if it's an existing repository.

Whoops, I added the wrong one.

git -r https://github.com/username/repository

And how do I get notified?

git -s to subscribe,
git -u to unsubscribe

That's all?

You can use git -h to see all available commands,
and more is coming!
